# Portfolio - Quentin Adeline


React + Tailwind + Framer Motion. Multilingue (FR/EN) et mode sombre.


## Installation
1. Copier les fichiers / placer les images dans public/assets
2. `npm install`
3. `npm run dev`


## Notes
- Langues en `public/lang/*.json`.
- La preference thème et langue sont conservées dans localStorage.
